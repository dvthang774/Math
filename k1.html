Phát biểu giả thiết:

H1: Phân phối của feature là không chuẩn

  • $\alpha$ mức ý nghĩa, significant level
  • $K_{tn}$ = giá trị stats
  • $u_{\alpha/2}$ được gọi là giá trị tới hạn (critical value)
  • $p$ value = xác suất $p=P(|X| > K_{tn})$ và $\alpha = P(|X|> u_{\alpha/2})=0.05$.
  • Miền bác bỏ $B_\alpha = (-\infty, -u_{\alpha/2})\cup (u_{\alpha/2}, + \infty)$.
  • Nếu $K_{tn} > u_{\alpha/2}$ (giá trị thực nghiệm > giá trị tới hạn) thì bác bỏ H0.

2. Kiểm định tính thuần nhất của phương sai (homogeneity of variance)¶

In [15]:
from scipy import stats
import numpy as np
def gen_sample(sigma=3, N=10000):
    x = np.arange(-10, 11)
    xU, xL = x + 0.5, x - 0.5 
    prob = stats.norm.cdf(xU, scale = sigma) - stats.norm.cdf(xL, scale = sigma)
    prob = prob / prob.sum() # normalize the probabilities so their sum is 1
    return np.random.choice(x, size = N, p = prob)

a) Bartlett’s Test¶

Phát biểu các giả thiết:

H0: Các feature đều có phương sai đồng nhất;

H1: Có ít nhất 2 feature có phương sai khác nhau (lớn).

Chú ý: Nếu $p<0.05=\alpha$ thì loại giả thiết H0. Tức là các feature không đồng nhất về phương sai.
