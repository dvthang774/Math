Z-test: So snh mean ca mu vi mean ca tng th. iu kin s dng Z-test:

  • Phng sai $\sigma$ bit v c mu $N$ ty .
  • Phng sai cha bit v $N\ge 30$.
  • Nu $N<30$ th cn kim tra d liu l phn phi chun.

Kiểm định 2 phía:¶

H0: $\mu = \mu_0$.

H1: $\mu \ne \mu_0$.

Vi mc ngha $\alpha$ (significant level), ta cn tm $t_*$ sao cho $P(|X| > t_*) = \alpha$. Gi tr $t_*$ c gi l gi tr ti hn (critical value).

Cch tnh $t_*$: Ta c $P(|X|\le t_*) = 1-\alpha$. Do tnh i xng, ta c $P(0\le X\le t_*) = P(-t_*\le X\le 0) = \frac{1-\alpha}{2}$. iu ny tng ng vi $\phi(t_*) = \frac{1-\alpha}{2}$.

V d: Vi $\alpha=0.05$ th $\frac{1-\alpha}{2} = 0.475$. Ta cn gii phng trnh $\phi(t_*) = 0.475$. T bng $A_2$ ta c $t_* = \phi^{-1}(0.475) = 1.96.$

V d: Vi $\alpha=0.01$ th $\frac{1-\alpha}{2} = 0.495$. Ta cn gii phng trnh $\phi(t_*) = 0.495$. T bng $A_2$ ta c $t_* = \phi^{-1}(0.495) = 2.57.$

Min bc b: $B_\alpha = (-\infty, -t_*)\cup (t_*, +\infty)$.

Gi $K_{tn}$ (statistical score) l gi tr thng k thc nghim c tnh ra t mu. Ta c gi tr $p$ ($p$ value) c tnh bi: $p = P(|X| > K_{tn})$.

Kt lun:

Nu $K_{tn}\in B_\alpha$ th bc b gi thit H0. iu ny tng ng vi $p<\alpha$.

Kiểm định 1 phía:¶

H0: $\mu = \mu_0$.

H1: $\mu > \mu_0$. (H1: $\mu < \mu_0$).

    std = np.std(data)

def results(p, vals=['mean1', 'mean2'], alpha=0.05):
    cols=['score', 'p_value', 'KetLuan']
    if p['p_value'] < alpha:
        p['KetLuan'] = f"Chp nhn H1 vi mc  ngha {alpha}"
    if p['p_value'] >= alpha:
        p['KetLuan'] = f"Chp nhn H0 vi mc  ngha {alpha}"
    df = pd.DataFrame(p, index=[''])
    if vals: cols = vals + cols
    return df[cols]
