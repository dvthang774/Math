Z-test: So sánh mean của mẫu với mean của tổng thể. Điều kiện sử dụng Z-test:

  • Phương sai $\sigma$ đã biết và cỡ mẫu $N$ tùy ý.
  • Phương sai chưa biết và $N\ge 30$.
  • Nếu $N<30$ thì cần kiểm tra dữ liệu là phân phối chuẩn.

Kiểm định 2 phía:¶

H0: $\mu = \mu_0$.

H1: $\mu \ne \mu_0$.

Với mức ý nghĩa $\alpha$ (significant level), ta cần tìm $t_*$ sao cho $P(|X| > t_*) = \alpha$. Giá trị $t_*$ được gọi là giá trị tới hạn (critical value).

Cách tính $t_*$: Ta có $P(|X|\le t_*) = 1-\alpha$. Do tính đối xứng, ta có $P(0\le X\le t_*) = P(-t_*\le X\le 0) = \frac{1-\alpha}{2}$. Điều này tương đương với $\phi(t_*) = \frac{1-\alpha}{2}$.

Ví dụ: Với $\alpha=0.05$ thì $\frac{1-\alpha}{2} = 0.475$. Ta cần giải phương trình $\phi(t_*) = 0.475$. Từ bảng $A_2$ ta có $t_* = \phi^{-1}(0.475) = 1.96.$

Ví dụ: Với $\alpha=0.01$ thì $\frac{1-\alpha}{2} = 0.495$. Ta cần giải phương trình $\phi(t_*) = 0.495$. Từ bảng $A_2$ ta có $t_* = \phi^{-1}(0.495) = 2.57.$

Miền bác bỏ: $B_\alpha = (-\infty, -t_*)\cup (t_*, +\infty)$.

Gọi $K_{tn}$ (statistical score) là giá trị thống kê thực nghiệm được tính ra từ mẫu. Ta có giá trị $p$ ($p$ value) được tính bởi: $p = P(|X| > K_{tn})$.

Kết luận:

Nếu $K_{tn}\in B_\alpha$ thì bác bỏ giả thiết H0. Điều này tương đương với $p<\alpha$.

Kiểm định 1 phía:¶

H0: $\mu = \mu_0$.

H1: $\mu > \mu_0$. (H1: $\mu < \mu_0$).

def critical_value(alpha, tail=0):
    '''Tính giá trị tới hạn: tail=0:: 2phia; tail=-1:left; tail=1:right'''
    if tail == -1: return norm.ppf(alpha)

import statsmodels.api as sm
# Đưa dữ liệu về dạng chuẩn tắc #
def standadize(data):
    mean = np.mean(data)
    std = np.std(data)
    return (data - mean)/std

sm.qqplot(standadize(data), line ='45')
plt.show()

def results(p, vals=['mean1', 'mean2'], alpha=0.05):
    cols=['score', 'p_value', 'KetLuan']
    if p['p_value'] < alpha:
        p['KetLuan'] = f"Chấp nhận H1 với mức ý nghĩa {alpha}"
    if p['p_value'] >= alpha:
        p['KetLuan'] = f"Chấp nhận H0 với mức ý nghĩa {alpha}"
    df = pd.DataFrame(p, index=[''])
    if vals: cols = vals + cols
    return df[cols]
